Frequently Asked Questions

The difference between var, let, and const
var is function-scoped and can be redeclared & updated, but it hoists with undefined.
let is block-scoped and can be updated but not redeclared in the same scope.
const is block-scoped and cannot be updated or redeclared after initialization.
The difference between map(), forEach(), and filter()
map() creates and returns a new array by applying a function to each element.
forEach() executes a function for each element but does not return a new array.
filter() creates and returns a new array with elements that pass a condition.
Explain arrow functions and how they are different from regular functions
Arrow functions (=>) are a shorter syntax for writing functions in JavaScript.
They don't have their own this, so they inherit this from the surrounding scope.
Unlike regular functions, they cannot be used as constructors and dont have arguments object.

How closures work in JavaScript
Closures allow a function to remember and access its outer scope even after the outer function has executed.
They help in data encapsulation by keeping variables private.
Used in callbacks, event handlers, and maintaining state in JavaScript.
